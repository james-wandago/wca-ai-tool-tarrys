# Beauty & Tattoo AI Assistant

AI-powered beauty and tattoo assistant for Tarrys Beauty Lounge.

## What it does
- Takes a client request (nails, lashes, tattoo, etc.)
- Uses Claude AI to recommend the best service
- Suggests a price range in Kenyan Shillings (KES)
- Generates a professional WhatsApp message for the client

## How to run
1. Install Python 3.10+
2. Install required package: `pip install anthropic`
3. Run: `python main.py`
4. Enter your Anthropic API key when prompted
5. Type a client request and press Enter
